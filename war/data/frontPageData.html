CTS2 XML to JSON Translator

This page will take XML and convert it to JSON using the proposed OMG XML to JSON Transformation Rules.

The XSLT transformation used by this application can be found HERE.
The project for this application is on GitHub.


Usage - You can perform one of the following actions:

  • Enter a REST service that returns XML. This will display the XML and converted JSON.
  • Enter some XML in the text area on the left and it will be transformed into JSON.
